# marketing-components

Marketing React components extracted from [data-visualization-studio](https://github.com/hgabrali/data-visualization-studio).

## Components

- **AEODashboard.tsx** - AEO Score dashboard with animated rings, metrics, share-of-model, and hallucination heatmap
- **CompetitorMatrix.tsx** - Competitor feature comparison matrix
- **ContactFormDialog.tsx** - Contact form dialog with Supabase integration
- **DemoSection.tsx** - Interactive demo section with tabbed panels
- **HowRankerCompares.tsx** - Comparison table component
- **KnowingVsWinning.tsx** - Two-column knowing vs winning section
- **LanguageSwitcher.tsx** - Language switcher dropdown (EN/DE/TR)
- **MarketingFooter.tsx** - Site footer with links and social icons
- **MarketingLayout.tsx** - Layout wrapper with nav and footer
- **MarketingNav.tsx** - Responsive navigation with auth integration
- **PillarCard.tsx** - Animated card with tilt/spotlight effects
- **SocialProofBar.tsx** - Stats bar with animated count-up
- **WhyRankerAI.tsx** - Why section with feature cards

## Tech Stack

- React / TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- react-i18next
- Lucide React
